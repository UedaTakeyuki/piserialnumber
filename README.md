# piserialnumber
Get Raspberry Pi board serialnumbermodel number

## install

```bash:
pip install getrpimodel
```

## return
String: Raspberry Pi board serialnumbermodel number string.

## How to use 
### as python program.

```bash:
python -m piserialnumber 
```

### as python library.

```python:
import getrpimodel

print (getrpimodel.serial())
```
